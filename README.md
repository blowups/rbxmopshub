# mopsHub
mopsHub is a Roblox exploit script for various Games

### Supported Games
- Arsenal
- Phantom Forces
- All Of Us Are Dead
- Military Simulator

### Features
## Arsenal
- Silent Aimbot
- Wallbang
- ESP
- Walkspeed Changer
- Infinite Jump
- Gun Mods (Inf Ammo, Rapidfire, ...)

## Phantom Forces
- Silent Aimbot
- Hit Part
- Hitbox Extender
- Chams ESP
- Infinite Jump
- Fly
- Auto Respawn

## All of Us Are Dead
- Kill All
- Gun Mods (Inf Ammo, Rapidfire, ...)

# Military Simulator
- Gun Mods (Inf Ammo, Rapidfire, ...)
